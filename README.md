## Folder structure (nested: a folder per week, a folder per day)
    week1/day1  week1/day2  week1/day3  week1/day4
    week2/day1  week2/day2  week2/day3  week2/day4  week2/day5

- day1 = the first homework day of that week (not a calendar date).
- Week 1 starts Monday (Sunday was orientation), so it has day1-day4.
- Week 2 runs Sunday-Thursday, so it has day1-day5.

## How to create a nested folder on GitHub
GitHub has no empty folders. Add file -> Create new file, then type the full
path in the filename box, e.g.  week1/day1/solution.py  -- the slashes create
the folders. Put the work inside and Commit.

## Deadlines
Each day's homework is due by 3:00 PM Istanbul time the NEXT workday
(Sunday-Thursday week). A Thursday assignment is due 3 PM the following Sunday.
After the deadline it is marked late automatically.

## Attendance is separate
Homework here; attendance through the daily Google Form. Do both every day.

Don't rename or delete past day-folders, and don't force-push over history.
