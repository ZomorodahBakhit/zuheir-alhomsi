# Zuheir Alhomsi - 88ninety Academy Internship

This is **your** private repository. Only you and the staff can see it.

## Where your homework goes
The folders are already created for you:

    week1/  day1  day2  day3  day4
    week2/  day1  day2  day3  day4  day5

- day1 = the first homework day of that week (not a calendar date).
- Week 1 starts Monday (Sunday was orientation), so it has day1-day4.
- Week 2 runs Sunday-Thursday, so it has day1-day5.

## Deadlines
Each day's homework is due by 3:00 PM Istanbul time the NEXT workday
(work week is Sunday-Thursday). A Thursday assignment is due 3 PM the following
Sunday. After the deadline it is marked late automatically.

## Attendance is separate
Homework here; attendance through the daily Google Form. Do both every day.

Don't rename or delete the day folders, and don't force-push over history.
